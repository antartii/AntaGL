◆ engine_should_close()

bool engine_should_close ( engine_t engine)

Returns a boolean if the window or application should close due to internal events, such as closing the window or any critical errors It is recommended to wait for this event to be true before cleaning up the engine with engine_cleanup. It is also strongly recommended to end your main loop upon this function returning false.

Parameters
enginePointer to the engine to check if the app should be closed
Returns
true if the application should close
false if the application's didn't had internal events telling it to close
